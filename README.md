# mdview

GTK application which does nothing but renders [Markdown] document onto the
screen, using WebKit.

## Requirements

- [GTKmm] 3.0
- [WebKitGTK2] 4.0

## How to build

```bash
mkdir build
cd build
cmake ..
make
```

[Markdown]: https://en.wikipedia.org/wiki/Markdown
[GTKmm]: https://gtkmm.org
[WebKitGTK2]: https://webkitgtk.org
